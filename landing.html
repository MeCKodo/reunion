import { IncomingMessage } from 'node:http'; import { verify } from 'jsonwebtoken'; export async function verifyAuth(req: IncomingMessage) { const token = extractBearerToken(req); if (!token) throw new AuthError('Missing token'); const payload = verify(token, process.env.JWT_SECRET!); return payload as JWTPayload; }

核心功能

为 AI Coding 工作流而生

从检索到导出，覆盖 AI 对话资产化的完整链路。

🔗

三大 Agent 聚合

自动识别 Cursor（旧 .txt + 新 .jsonl 双格式）、Claude Code、Codex CLI 的本地会话数据，统一索引。

⚡

秒级全文检索

中英文混合搜索，支持 6 档时间范围、项目过滤、Source 过滤，命中片段角色标注与高亮预览。

📖

结构化对话视图

将原始 transcript 解析为 User / AI / Tool / Subagent 分类视图，代码块 Dracula 着色，Tool 调用按 Read / Write / Exec 分组。

🤖

AI Smart 导出

接入 OpenAI / Cursor Agent，用 AI 将对话精华提炼为 RULES.md 或 SKILL.md，一键写入项目目录。

🏷️

星标与标签系统

对话加星标收藏，自定义标签分类管理（#refactor, #debug, #design），标签 OR 匹配快速筛选。

🖥️

macOS 原生桌面体验

Electron 桌面应用，交通灯窗口控件，数据纯本地不外传。支持 Apple Silicon 和 Intel 双架构。

Smart Export

对话即知识
一键沉淀到项目

接入 AI Provider，将对话中的经验、规范、技巧提炼为标准文件，直接写入你的项目目录。

下次 AI 会话加载这些文件，就能继承之前的经验——形成知识飞轮。

RULES.md编码规范

从对话中提取架构决策、命名约定、技术选型理由，生成可版本控制的编码规范文件。

SKILL.md技能文件

将调试技巧、工作流程、最佳实践提炼为 Agent Skill，后续 AI 会话直接加载复用。

Multi-Provider灵活切换

支持多个 ChatGPT 账号 OAuth + Cursor Agent，Settings 面板统一管理，自动路由到默认 Provider。
